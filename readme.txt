Program that grabs and dumps configs for cisco ios devices.

Requires Python 3.12 (LTS Microsoft Store Works!)
Install requirements by running start.bat
OR
Install requirements by opening a terminal in the directory and running:
   pip install -r requirements.txt


Reads from text files to populate a list to grab the configs. 
Format is as such:
10.0.250.0 (Enter)
10.0.250.1
10.0.250.2

Do not add the extension to the text file when selecting in the program...
Read vlans.txt and this link for info on adding switch models and vlans. (https://www.w3schools.com/python/python_dictionaries_nested.asp)
